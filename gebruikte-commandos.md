## SSH access

### commands
```
ssh-keygen             # Step 1: Public and Private Key Generation

chmod 600 ~/.ssh/id_rsa       # Step 2:  Change Permissions

ssh-copy-id user@IP    # Step 3:  Configure Server

ssh user@IP            # step 4:  Verify access
```
